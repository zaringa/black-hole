# Black Hole Simulation (PyVista)

Проект визуализации черной дыры и аккреционного диска на `PyVista`.

## Что улучшено

- Добавлена физически более правдоподобная кинематика диска.
- Дифференциальное вращение по радиусу (`Omega(r)`, а не одинаковая скорость для всего диска).
- Радиальный дрейф вещества внутрь (модель вязкого аккреционного потока).
- Псевдо-ньютоновский потенциал Пачиньского-Вииты для поведения рядом с горизонтом.
- Температурный профиль тонкого диска с усилением яркости по азимуту.
- Джеты теперь пульсируют в зависимости от уровня аккреции.
- Добавлены орбитальные частицы, вращающиеся вокруг черной дыры в плоскости диска.
- Добавлена планета, которая орбитально падает к черной дыре и разрушается на фрагменты перед поглощением.
- Усилено поэтапное разрушение планеты (ранний разрыв, более интенсивная фрагментация).
- Добавлена модель Солнца: многослойная температура (внутри горячее), протуберанцы и выбросы частиц с цветом по температуре.
- Если Солнце входит в горизонт событий, включается захват: оно ускоренно тянется к центру черной дыры и поглощается.
- Добавлено приливное разрушение Солнца: звезда деформируется, сжимается и распадается на фрагменты при сближении с черной дырой.

## Запуск

```bash
make start
```

Или напрямую:

```bash
python3 -m venv venv
./venv/bin/pip install -r requirements.txt
./venv/bin/python main.py
```

На выходе создается `black_hole_animation.gif` (в него пишутся только первые секунды анимации). Интерактивная анимация в окне после запуска не останавливается и идет до закрытия окна.

## Расчеты по формулам

Все основные формулы (Schwarzschild, Kerr, Hawking, Eddington, redshift, tidal, ISCO) реализованы в `black_hole_lib.py` в SI-единицах.

Демо минимального набора выходных величин:

```bash
make gptvers
```
